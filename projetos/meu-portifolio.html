Site Pessoal Profissional

Desenvolvimento de um site institucional para apresentação profissional, com foco em organização, clareza e escalabilidade.

Contexto

Antes do projeto, não havia uma presença digital clara que representasse minhas habilidades técnicas e minha forma de pensar como desenvolvedor. Currículos tradicionais não comunicavam bem meu perfil.

Objetivo

Criar um site profissional que reunisse projetos, habilidades e valores, servindo como base sólida para entrevistas e oportunidades no mercado de tecnologia.

Solução Desenvolvida

Desenvolvi um site utilizando HTML semântico para estruturação, CSS para responsividade e organização visual, e JavaScript para interações básicas. Todas as decisões priorizaram clareza, manutenibilidade e experiência do usuário.

Tecnologias Utilizadas

  • HTML5 (estrutura semântica)
  • CSS3 (layout responsivo)
  • JavaScript (interações)
  • Git & GitHub (versionamento)

Resultado e Aprendizados

O projeto consolidou boas práticas de organização de código, separação de responsabilidades e comunicação técnica. Além disso, tornou-se uma base escalável para novos projetos e melhorias futuras.

Próximos Passos

Implementar melhorias visuais, otimizações de performance, acessibilidade e novas funcionalidades conforme evolução técnica.
